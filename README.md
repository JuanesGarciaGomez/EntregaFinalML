# Regresión del Volumen de Tráfico — Machine Learning

Proyecto final de Machine Learning: regresión del volumen de tráfico vehicular usando siete modelos con cuatro métodos de optimización de hiperparámetros y un modelo híbrido GRA-XGBoost.

## Ver el Jupyter Book

[https://JuanesGarciaGomez.github.io/EntregaFinalML](https://JuanesGarciaGomez.github.io/EntregaFinalML)

## Instalar dependencias

```bash
pip install -r requirements.txt
```

## Modelos implementados

- XGBoost, Random Forest, KNN, Árbol de Decisión, SVR, Ridge, Lasso
- Métodos de optimización: Grid Search, Random Search, Bayesiana (Optuna), Algoritmo Genético (DEAP)
- Modelo híbrido: GRA-XGBoost (Grey Relational Analysis + XGBoost)
- Análisis estadístico: Test DM, Wilcoxon, Bootstrap IC 95%, LIME

## Dataset

Metro Interstate Traffic Volume — UCI Machine Learning Repository.
